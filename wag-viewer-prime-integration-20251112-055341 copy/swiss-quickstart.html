What this environment is good for
Swiss – choose builders Frank – assemble scenes Grace – view and edit

The Swiss / Frank / Grace studio is a LEGO story factory:

  • Pick shells, locations, vehicles and minifigs with the -ators.
  • Send MPD scenes over the wag-frank bus into Frank.
  • Open the same scenes in Grace to study, edit and iterate.

Every builder (-ator) produces real MPD text or parts that flow into a single wag-frank-scene.mpd you can keep, share, or paste into a language model for remixing.

Hello world flow – place a minifig in a scene
  1. In Swiss, go to SCENEERATOR. Pick a shell you like (crater, road grid, harbor hull) and click "Load Scene Shell". This sends a starter MPD to Frank and downloads a copy.
  2. Still in Swiss, go to MINIFIGURATOR. Build a simple figure (any hat / head / torso / legs). Use "Save MPD" to export the minifig to Frank.
  3. Open the Swiss / Frank / Grace studio tab. Switch to FRANK and look for your scene shell with the minifig cells listed in the grid.
  4. Switch the studio to GRACE. Load the current wag-frank scene and orbit the camera – you should see your shell and minifig on the same stage.

Once this loop works, you have the basic pipeline: builder → Frank → Grace.

Next challenges
  • Use LOCATIONATOR to add walls, doors, windows and props to the same shell. Send the updated location MPD to Frank, then refresh Grace.
  • In VEHICULATOR, design a small car or plane and export it as an MPD so Frank can park it in the scene.
  • Open MOVIEATOR or WORLDINATOR to pull in reference scenes and worlds – use their Load / Add Scene buttons to seed Frank with example layouts.
Working with language models

Grace and Frank always keep the MPD text visible. To collaborate with a language model on layout:

  • Open the current wag-frank-scene.mpd in Frank or Grace.
  • Select the region you care about (for example a single cell or a group of parts around a character).
  • Paste that MPD text into a model with a prompt like "re-layout this street for a night chase, keep the minifigs and overall scale".
  • Paste the rewritten MPD back into Grace and reload the scene.

The goal is to make MPD feel like editable source code for scenes, not a locked binary file.

Mental model – how to navigate
  • Use the dots in Swiss to switch between designers (MINIFIG, LOCATION, VEHICLE, SCENE, MOVIE, WORLD, DOCS).
  • Use the studio tabs to switch between SWISS, FRANK and GRACE views of the same project.
  • When in doubt, start here, then jump to a builder from the dots.
