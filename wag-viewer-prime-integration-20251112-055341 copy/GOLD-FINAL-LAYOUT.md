## Session Management (Bronze Style)

**Vertical Dots:**
- Each dot = one scene/session
- Gold dot = active
- Gray dots = inactive
- Hover = semi-gold preview
- Click = switch instantly

**New Session:**
- Click `+` button
- Creates blank scene
- Auto-switches to it
- Adds new dot to strip

**Benefits:**
- Work on multiple MPDs simultaneously
- Quick switching (no dropdown needed)
- Visual session indicator
- Can have 10+ scenes easily
